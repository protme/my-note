
✨利用 Cloudflare 的免费全家桶Pages，D1数据库和 Workers，再加上现在最强的AI模型，实现加密NOTE

1️⃣Cloudflare存储与数据库---→D1 SQL数据库---→创建数据库---→名字随便写

找到数据库的控制台，，把SQL 文件代码复制粘贴然后点击执行，看到执行成功，点击右上角的探索数据 DATA
左边一个notes 表

2️⃣回到主页，点击 计算和AI，选择 Workers 和 Pages
创建应用程序，从 Hello World 开始，名字随便写，然后再点击部署
部署完成之后，绑定两个功能
D1数据库，变量名称填 DB 
再添加一个绑定，选择 Workers AI
变量名称填 AI 
然后点击绑定

尽量设置一个自定义的域名

找到变量和机密，类型选择密钥
变量名称： ADMIN_KEY
密码随便写

打开Workers 的 JS 文件
代码复制粘贴到
Workers 的编辑框，点击部署

3️⃣回到 Cloudflare 的 Workers 和 Pages
点击创建 Pages 导入现有的 Git 存储库
名字默认就可以，构建命令、目录留空，直接保存并部署
给 Pages 设一个自定义域名
打开 Pages 的链接，第一次进来的时候
会显示一个初始化的配置，需要两个东西
1.Workers 的链接，加前缀 https:// 末尾不加 /
2.密钥，就是环境变量里设置的密码

4️⃣这个系统现在可以使用了，第一步为所有的笔记设置一个密码
然后特别隐私的那些数据设置一个单独的密码

然后就可以写内容了
比如：大家好，然后点击加密存入云端
这个时候它就会先在本地加密生成乱码，然后再传出去
然后我们再到 D1数据库 后台来看一下，选择这个 notes 这个表
点击这里的刷新
就可以看到这个笔记的内容了
